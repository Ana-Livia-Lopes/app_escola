# Sistema Mobile de Controle de Turmas e Atividades — SAEP 2023.1

## Contexto

Este projeto foi desenvolvido com o objetivo de criar um aplicativo mobile para controle de turmas e atividades de professores, inspirado no SAEP 2023.1.  

Em muitas escolas, especialmente em regiões remotas do Brasil, a falta de sistemas organizacionais pode dificultar o acompanhamento das atividades desenvolvidas pelos professores.  
Com este aplicativo, o docente pode realizar login, cadastrar e visualizar suas turmas, além de registrar e gerenciar as atividades associadas a cada uma delas.

---

## Instruções para Uso

1. **Instale as dependências do projeto**
   ```bash
   npm install
   ```

2. **Execute o aplicativo com o Expo**
   ```bash
   npx expo start --tunnel
   ```

3. **Abra o aplicativo no celular**
   - Escaneie o QR Code exibido no terminal usando o aplicativo **Expo Go**.
